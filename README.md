# 小红书链接自动识别与跳转


一个面向 `xiaohongshu.com` 的 Tampermonkey 用户脚本。

它可以自动识别小红书网页中帖子正文和评论区里的纯文本链接，并将其转换为可点击链接；同时会在帖子弹窗的关注按钮左侧插入一个更协调的 `直达` 按钮，帮助你更方便地打开当前笔记页面。

![Preview](./assets/preview.svg)


## 功能特点

- 自动识别帖子正文中的纯文本链接
- 自动识别评论区中的纯文本链接
- 将纯文本链接转换为可点击链接
- 链接默认在新标签页中打开
- 自动清理常见分享文案中的冗余引导文本
- 在帖子弹窗中插入 `直达` 按钮
- 适配小红书网页的动态加载内容
- 优化按钮样式，使其和页面原生按钮更协调

## 适用场景

- 查看别人分享的小红书内容时，正文中有纯文本网址，希望直接点击访问
- 浏览评论区时，希望快速打开用户贴出的链接
- 在笔记弹窗中，希望直接跳转当前笔记，而不是自己复制地址
- 希望减少手动复制链接、切换页面的操作成本


## 安装方法

### 方法一：手动安装

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开 Tampermonkey 控制面板。
3. 点击“创建新脚本”。
4. 删除默认模板内容。
5. 将脚本全部内容粘贴进去。
6. 保存脚本。
7. 打开或刷新小红书网页进行测试。

### 方法二：通过 GitHub Raw 安装

当你把仓库上传到 GitHub 后，可以打开脚本文件的 Raw 链接，Tampermonkey 通常会自动识别并进入安装流程。

## 使用方法

1. 确保脚本已在 Tampermonkey 中启用。
2. 打开 `https://www.xiaohongshu.com/`。
3. 浏览帖子详情页或评论区。
4. 当页面中出现纯文本 URL 时，脚本会自动将其转换成可点击链接。
5. 当你打开帖子弹窗时，关注按钮左侧会出现 `直达` 按钮。
6. 点击 `直达` 按钮，即可在新标签页打开当前笔记页面。

## 当前行为说明

- 脚本只在小红书网页端生效
- 链接默认在新标签页打开
- `直达` 按钮目前默认打开当前页面 URL
- 主要处理区域为帖子正文、评论区和帖子弹窗

- Brave
- 其他兼容 Tampermonkey 的浏览器

## 更新与优化

当前版本：`1.0.0`

这个版本相较于早期脚本，已经完成以下优化：

- 将文本替换逻辑改为 DOM 安全构造，避免直接拼接 `innerHTML`
- 使用增量监听机制处理动态加载内容，减少全页重复扫描
- 使用 `TreeWalker` 处理文本节点，降低重复处理和性能开销
- 增加去重机制，减少重复插入按钮和重复转换链接的问题
- 优化 `直达` 按钮外观，使其在页面中更自然、更协调
- 重构脚本结构，提升可读性和可维护性
